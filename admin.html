How It Works

1. Edit any section using the left sidebar navigation.
2. Click Save Draft to store your changes in this browser.
3. Go to Deploy to Site and click Deploy to GitHub to publish live.
4. Your website updates within 60 seconds after deploying.

All content is stored in content.json — the single source of truth for your entire website.
